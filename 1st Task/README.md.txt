Bank Marketing: Term Deposit Prediction
Objective

Predict whether a bank customer will subscribe to a term deposit using machine learning models.

Dataset

Bank Marketing Dataset (UCI Machine Learning Repository)

Workflow
Loaded and explored dataset
Encoded categorical features
Trained classification models:
Logistic Regression
Random Forest
Evaluated models using:
Confusion Matrix
F1-Score
ROC Curve
Used SHAP/LIME to explain at least 5 predictions
Key Skills
Classification modeling
Feature encoding
Model evaluation
Explainable AI (SHAP/LIME)
Tools Used

Python, Pandas, Scikit-learn, Matplotlib, Seaborn, SHAP, LIME

Outcome

Built models to predict customer subscription behavior and explained predictions for better interpretability.
